# React Django

Frontend for the django-api repository. Components used are from Material UI.

# .env config 

  REACT_APP_API_URL=http://localhost:8000/api

# Usage

  1. Install dependencies

  ```
    npm install

    # or 

    yarn install
  ```

  2. Run Application
  
  ``` 
    npm start

    or 

    yarn start
  ```
